# Tableau 可视化看板

由于 Tableau Public 发布限制，本项目通过截图展示可视化成果。
三张看板截图位于 `../images/` 文件夹。

## 看板一览

| 看板 | 文件名 | 内容 |
|------|--------|------|
| 看板一：销售总览 | dashboard1_sales_overview.png | KPI总览、月度趋势、品类占比、地区对比 |
| 看板二：产品分析 | dashboard2_product.png | 子品类利润、Top10排行、折扣与利润关系 |
| 看板三：用户画像 | dashboard3_customer.png | RFM分层、客户贡献、地理分布、热力图 |
