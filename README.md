# Superstore 销售数据分析

> 🎓 上海对外经贸大学 | 数据科学与大数据技术 | 李诗甜

## 项目背景

使用 Superstore 数据集（约 10,000 条零售订单记录），模拟零售企业数据分析场景，从**整体趋势、产品表现、地区差异、用户分层**四个维度进行深入分析，发现业务问题和增长机会。

## 技术栈

| 技术 | 用途 |
|------|------|
| **MySQL** | 数据存储、SQL 查询分析（CTE、窗口函数、RFM 模型） |
| **Tableau** | 可视化看板搭建（3 个仪表板） |
| **Python** | 数据导入与预处理 |

## 分析框架

```
整体概览 → 产品维度 → 地区维度 → 用户分层(RFM) → 时间趋势
  │            │           │           │              │
 KPI总览    品类利润   区域排名    客户价值分类   季节性分析
```

## 核心发现

### 📊 看板一：销售总览
![销售总览](images/dashboard1_sales_overview.png)

> **关键结论：**
> - 总销售额 **229.72 万**，总利润 **28.64 万**，整体利润率 **12.47%**
> - 销售额呈逐年上升趋势
> - Technology 品类销售额占比最高（约 36.4%）

---

### 📦 看板二：产品分析
![产品分析](images/dashboard2_product.png)

> **关键结论：**
> - **Tables** 子品类亏损最为严重（**−17,725 元**），Bookcases（−3,473 元）、Supplies（−1,189 元）也处于亏损状态
> - 销售额 Top 10 产品 ≠ 利润 Top 10 产品，高销量不代表高利润
> - 折扣与利润呈明显负相关，建议严格控制高折扣订单审批

---

### 👤 看板三：用户画像（RFM 分析）
![用户画像](images/dashboard3_customer.png)

> **关键结论：**

| 客户分层 | 客户数 | 占比 | 贡献销售额 | 销售额占比 |
|----------|--------|------|-----------|-----------|
| 高价值客户 | 191 | 24.1% | 87.56 万 | **38.1%** |
| 低价值客户 | 371 | 46.8% | 67.30 万 | 29.3% |
| 流失风险客户 | 97 | 12.2% | 48.05 万 | 20.9% |
| 重点发展客户 | 134 | 16.9% | 26.80 万 | 11.7% |

> - 近 **1/4 的高价值客户贡献了近 4 成销售额**，是核心资产
> - **流失风险客户 97 人**（消费高但长期未回购），建议定向发放优惠券召回
> - 低价值客户数量最多（46.8%）但贡献不到 3 成，可降低营销投入

## SQL 分析模块

| 模块 | 内容 | 核心 SQL 技术 |
|------|------|--------------|
| 整体概览 | 总销售额/利润/利润率、年度趋势 | GROUP BY, 聚合函数 |
| 产品分析 | 品类/子品类利润、亏损产品、折扣分析 | HAVING, CASE WHEN |
| 地区分析 | 区域对比、Top/Bottom 州、交叉分析 | 多维度 GROUP BY |
| RFM 用户分层 | R/F/M 计算、NTILE 打分、四层分类 | CTE, NTILE, 窗口函数 |
| 时间趋势 | 月度趋势、季度对比、环比增长 | LAG, DATE_FORMAT |
